## SCR-001: ログイン画面

### 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-001 |
| 画面名 | ログイン |
| アクター | 全ユーザー |
| 概要 | メールアドレスとパスワードでシステムにログインする |

### 画面レイアウトイメージ

```
┌──────────────────────────────────┐
│        営業日報システム            │
│                                  │
│  メールアドレス: [____________]   │
│  パスワード:     [____________]   │
│                                  │
│         [ ログイン ]              │
│                                  │
└──────────────────────────────────┘
```

### 画面項目一覧

| No. | 項目名 | 項目ID | 種別 | 必須 | 備考 |
|-----|--------|--------|------|------|------|
| 1 | メールアドレス | email | テキスト入力 | ○ | メール形式バリデーション |
| 2 | パスワード | password | パスワード入力 | ○ | マスク表示 |
| 3 | ログインボタン | btn_login | ボタン | - | 認証処理を実行 |

### バリデーション

| No. | 条件 | エラーメッセージ |
|-----|------|------------------|
| V-1 | メールアドレスが未入力 | メールアドレスを入力してください |
| V-2 | メールアドレスの形式が不正 | 正しいメールアドレスを入力してください |
| V-3 | パスワードが未入力 | パスワードを入力してください |
| V-4 | 認証失敗 | メールアドレスまたはパスワードが正しくありません |

### 画面イベント

| No. | イベント | トリガー | 処理内容 | 遷移先 |
|-----|----------|----------|----------|--------|
| E-1 | ログイン | ログインボタン押下 | 認証API呼出。成功時セッション発行 | SCR-010 |

## SCR-010: 日報一覧画面

### 基本情報

| 項目 | 内容 |
|------|------|
| 画面ID | SCR-010 |
| 画面名 | 日報一覧 |
| アクター | 営業担当者・上長 |
| 概要 | 日報を検索・一覧表示する。ログイン後のホーム画面を兼ねる |

### 画面レイアウトイメージ

```
┌──────────────────────────────────────────────────┐
│ [≡メニュー]  営業日報システム     山田太郎 ログアウト│
├──────────────────────────────────────────────────┤
│ ■ 日報一覧                      [ + 日報を作成 ] │
│                                                  │
│ 検索条件:                                         │
│  報告日: [____/__/__] 〜 [____/__/__]             │
│  営業担当者: [プルダウン▼]                         │
│                        [ 検索 ] [ クリア ]        │
│──────────────────────────────────────────────────│
│ No. | 報告日     | 営業担当者 | 訪問件数 | コメント数│
│  1  | 2026-02-24 | 山田太郎   |    3    |    2     │
│  2  | 2026-02-23 | 山田太郎   |    2    |    0     │
│  3  | 2026-02-24 | 佐藤花子   |    1    |    1     │
│                                                  │
│              < 1 2 3 ... 10 >                    │
└──────────────────────────────────────────────────┘
```

### 画面項目一覧

**検索条件エリア**

| No. | 項目名 | 項目ID | 種別 | 必須 | 備考 |
|-----|--------|--------|------|------|------|
| 1 | 報告日（From） | search_date_from | 日付入力 | - | カレンダーピッカー |
| 2 | 報告日（To） | search_date_to | 日付入力 | - | カレンダーピッカー |
| 3 | 営業担当者 | search_sales_person | プルダウン | - | 営業マスタから取得。上長は全員表示、担当者は自分のみ |
| 4 | 検索ボタン | btn_search | ボタン | - | 検索実行 |
| 5 | クリアボタン | btn_clear | ボタン | - | 検索条件を初期化 |
| 6 | 日報作成ボタン | btn_create | ボタン | - | 日報作成画面へ遷移 |

**一覧エリア**

| No. | 項目名 | 項目ID | 種別 | 備考 |
|-----|--------|--------|------|------|
| 7 | No. | row_no | 表示 | 連番 |
| 8 | 報告日 | report_date | 表示 | YYYY-MM-DD形式 |
| 9 | 営業担当者 | sales_person_name | 表示 | 氏名表示 |
| 10 | 訪問件数 | visit_count | 表示 | 訪問記録の件数 |
| 11 | コメント数 | comment_count | 表示 | コメントの件数 |

### 表示ルール

- 初期表示時、ログインユーザーの直近1週間分の日報を表示する
- 上長ロール（役職が「課長」以上）は全営業担当者の日報を閲覧可能
- 一般営業担当者は自分の日報のみ閲覧可能
- 1ページあたり20件表示、ページネーションあり
- 報告日の降順でソート

### 画面イベント

| No. | イベント | トリガー | 処理内容 | 遷移先 |
|-----|----------|----------|----------|--------|
| E-1 | 検索 | 検索ボタン押下 | 検索条件で日報一覧を取得 | - (再描画) |
| E-2 | クリア | クリアボタン押下 | 検索条件を初期値に戻す | - (再描画) |
| E-3 | 新規作成 | 日報作成ボタン押下 | - | SCR-011 (新規) |
| E-4 | 行クリック | 一覧行クリック | - | SCR-012 |

|------------|--------|----------|

| 顧客マスタ | SCR-020 | 上長（管理者）のみ |

| 役職 | ロール | 日報閲覧 | 日報作成・編集 | コメント投稿 | マスタ管理 |
|------|--------|----------|--------------|------------|----------|
| 担当/主任 | 一般 | 自分のみ | 自分のみ | ○ | × |
| 係長以上 | 上長 | 全員分 | 自分のみ | ○ | ○ |
